# 项目交接说明

## 一、项目概述

**项目名称**：政府采购招投标销售情报系统
**定位**：基于 Agentic RAG 架构的智能销售分析平台，帮助销售人员快速检索招标项目、获取客户画像、AI 生成销售建议。

**核心技术**：
- Agentic RAG（Agentic Retrieval-Augmented Generation）
- FAISS + BM25 混合检索
- LLM 大模型（DeepSeek-V3 / Qwen3.5-27B）
- FastAPI Web 服务

## 三、数据流向

```
1. 数据采集
   爬虫采集 → 原始数据 (data/raw/)

2. ETL 处理
   原始数据 → LLM 抽取 → 结构化数据 (data/output/etl/*.jsonl)

3. 索引构建
   结构化数据 → 向量化 + BM25 → FAISS 索引 (data/vector_store/)

4. 检索服务
   产品检索请求 → DualRetriever → 混合检索 → 返回结果

5. AI 分析
   招标数据 + 客户画像 → SalesAdvisor (LLM) → 销售建议
```

## 四、核心模块说明

### 4.1 Agentic RAG（src/agent/）

基于 Skill 插件架构的 Agent，支持多轮对话和自动工具调用。

**关键类**：
- `SalesAgent`：主 Agent 类，`run()` 方法执行多轮对话
- `SkillRegistry`：Skill 注册中心，管理所有可用工具
- `BaseSkill`：所有 Skill 的基类，定义 `name`、`description`、`parameters`、`execute()`

**已注册的 Skills**：
| Skill 名称 | 功能 |
|------------|------|
| `search_tenders` | 招标项目搜索 |
| `get_customer_profile` | 获取客户画像 |
| `list_customers` | 获取客户列表 |
| `generate_sales_suggestions` | AI 生成销售建议 |
| `analyze_project` | 项目机会分析 |

**Agent 初始化**：
```python
from src.agent import SalesAgent, create_agent
agent = create_agent(verbose=True)
```

---

### 4.2 检索模块（src/retrieval/retriever.py）

`DualRetriever` 类实现 FAISS 向量检索 + BM25 关键词检索的混合检索。

**核心方法**：
```python
retriever = DualRetriever(data_type="tender")
results = retriever.hybrid_search(
    query_text,
    top_k=10,
    use_vector=True,
    use_bm25=True,
    vector_weight=0.5,
    bm25_weight=0.5,
    province=None,
    aggregate_by_project=True,
    exclude_won=False,
    sort_by="score"
)
```

---

### 4.3 销售建议引擎（src/analysis/sales_advisor.py）

`SalesAdvisor` 类通过 LLM 生成销售切入点建议，支持 JSONL 缓存避免重复调用。

**核心方法**：
```python
advisor = SalesAdvisor(use_cache=True)
suggestions = advisor.generate_suggestions(
    project_data={"project_name_std": "...", ...},
    customer_profile={...},
    customer_name="客户名"
)
```

**缓存路径**：`data/output/sales_suggestions/suggestions.jsonl`
**缓存策略**：基于 `buyer_name + project_name` 的 MD5 哈希作为 Key

## 七、常见问题

### Q1：检索无结果
- 检查 FAISS 索引是否存在（`data/vector_store/`）
- 检查 `src/config.py` 中 `DATA_DIR` 路径是否正确

### Q2：AI 服务不可用
- 确认 LLM 服务（10.210.10.51:8001）是否运行
- 检查 `LOCAL_LLM_CONFIG` 配置

### Q3：客户画像为空
- 确认 `data/output/customer/customer_profiles.jsonl` 是否存在
- 可重新运行 `python src/analysis/generate_customer_profiles.py`

### Q4：销售建议重复生成
- 系统已内置缓存，相同项目+客户会复用缓存
- 缓存路径：`data/output/sales_suggestions/suggestions.jsonl`
