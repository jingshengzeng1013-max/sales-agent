# 项目详细流程说明

## 整体架构

```
数据爬取 → 数据存储 → 结构化抽取 → 检索/查询 → 销售使用
```

## 详细执行步骤

### 步骤 1: 爬取列表页
```bash
python src/crawler/ccgp_crawler.py
```
- 访问中国政府采购网搜索页面
- 搜索关键词（在 config.py 中配置，默认"通信"）
- 爬取指定页数（可配置，默认 100 页）
- 去重后存入数据库
- 支持代理轮换

### 步骤 2: 爬取详情页
```bash
python src/crawler/crawl_detail.py
```
- 读取数据库中 status='new' 的记录
- 访问每条招标的详情 URL
- 提取正文内容、附件链接
- 更新数据库 content 字段
- 异常内容 HTML 保存到 html/ 目录

### 步骤 3: 结构化抽取
```bash
# 全量抽取
python src/etl/extract_structured.py --all

# 限制数量测试
python src/etl/extract_structured.py --limit 50

# 导入到数据库
python src/etl/import_structured_db.py
```
- 规则抽取：公告类型、预算、地区等
- LLM 抽取：产品关键词、机会评分等
- 结果存入 tender_structured 表

### 步骤 4: 项目聚合
```bash
python src/etl/aggregate_projects.py
```
- 标准化项目名称，去除公告类型后缀
- 按采购单位分组
- 识别并聚合同一项目的多条公告
- 输出 projects_aggregated.jsonl

### 步骤 5: 生成客户画像
```bash
python src/analysis/generate_customer_profiles.py
```
- 按采购单位聚合历史招标
- 生成 360° 客户画像
- 输出 customer_profiles.jsonl

### 步骤 6: 向量化和索引构建
```bash
# 向量化
python src/vectorization/vectorize_data.py --type tender

# 构建索引
python src/vectorization/build_index.py --type tender
```

### 步骤 7: 启动服务
```bash
python webapp/server_fastapi.py
```
- 访问 http://127.0.0.1:8103/static/demo.html

## 数据流向

```
政府采购网
    │
    ├── ccgp_crawler.py → tenders 表（列表数据）
    │                          │
    │                          └── crawl_detail.py → tenders 表（详情数据）
    │                                                      │
    │                                                      └── extract_structured.py → tender_structured 表
    │                                                                              │
    │                                                                              ├── aggregate_projects.py → projects_aggregated.jsonl
    │                                                                              │
    │                                                                              └── generate_customer_profiles.py → customer_profiles.jsonl
    │
    └── vectorize_data.py → 向量数据 → build_index.py → FAISS/BM25 索引
                                                                    │
                                                                    └── server_fastapi.py → demo.html 前端展示
```
