## 详细执行步骤

### 步骤 1: 爬取列表页
```bash
python src/crawler/ccgp_crawler.py
```
- 访问中国政府采购网搜索页面
- 搜索关键词："芯片"
- 爬取指定页数（默认 3 页，每页约 20 条）
- 去重后存入数据库
- 随机延迟 2-5 秒，避免被封 IP

### 步骤 2: 爬取详情页
```bash
python src/crawler/crawl_detail.py [数量]
```
- 读取数据库中 status='new' 的记录
- 访问每条招标的详情 URL
- 提取正文内容、附件链接
- 更新数据库 content 字段
- 异常内容 HTML 保存到 html/ 目录

### 步骤 3: 结构化抽取
```bash
# 纯规则抽取
python src/etl/extract_structured.py --all

# 使用 LLM 增强抽取
python src/etl/extract_structured.py --all --use-llm
```
- 规则抽取：公告类型、预算、地区等
- LLM 抽取：产品关键词、机会评分等
- 结果存入 tender_structured 表

### 步骤 4: 查询使用
```bash
python src/etl/query_tenders.py
```
- 查看招标列表
- 关键词搜索
- 统计信息
- 导出 JSON

## 数据流向

```
政府采购网 → 爬虫 → tenders 表 → 抽取脚本 → tender_structured 表 → 查询/前端
                              ↓
                        html/ (异常备份)
```
