## 报告说明

1. **语义评分标准**: 基于关键词与中科晶上产品线的对应程度，10分=直接对应旗舰产品，1分=过于宽泛或无关
2. **数量评分标准**: 基于CCGP一年全文检索结果条数，10分=100条以上，0分=无结果
3. **综合评分**: 语义评分×40% + 数量评分×60%，反映关键词在政采场景的实际价值
4. **数据说明**: 68个关键词全部测试，5个关键词因网络问题未完成测试（结果为0或缺省）
