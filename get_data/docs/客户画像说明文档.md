# 客户画像说明文档

## 一、数据源

| 数据源 | 客户数 | 说明 |
|--------|--------|------|
| **招标数据** | 311个 | 政府/事业单位客户，基于政府采购招标历史 |
| **CRM数据** | 308个 | 企业客户，包含商机(93条)和线索(349条) |
| **合计** | **619个** | 全部客户 |

### 1.1 招标数据来源

- 政府采购招标公告（ccgp.gov.cn）
- 包含：采购单位、预算金额、技术要求、产品关键词等

### 1.2 CRM数据来源

| CRM数据 | 记录数 | 说明 |
|---------|--------|------|
| 商机列表 | 93条 | 进行中/关闭/签约状态的商机 |
| 线索列表 | 349条 | 跟进中/已转化/已放弃的线索 |

## 六、使用场景

### 6.1 销售跟进

```python
# 获取某客户的所有商机和线索
profile = load_profile("比亚迪")
print(f"商机数: {profile['value_profile']['opportunity_count']}")
print(f"线索数: {profile['value_profile']['lead_count']}")
for opp in profile['crm_history']['opportunities']:
    print(f"  {opp['opportunity_name']} - {opp['followup_status']}")
```

### 6.2 客户筛选

```python
# 筛选有进行中商机的客户
profiles = load_all_profiles()
active_opps = [p for p in profiles
               if p['value_profile']['opportunity_count'] > 0
               and any(o['opportunity_status'] == '进行中'
                       for o in p['crm_history']['opportunities'])]
```

### 6.3 销售对接

```python
# 获取某客户的销售负责人
profile = load_profile("比亚迪")
sales = profile['sales_relationship']['primary_sales']
print(f"主销售: {sales}")
```
