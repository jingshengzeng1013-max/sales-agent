# 模块开发教程

本教程指导开发人员如何为项目添加新的功能模块。

```
需求分析 → 设计方案 → 编码实现 → 单元测试 → 集成测试 → 文档更新 → 代码审查
```

## 二、示例：添加新的数据分析模块

### 2.1 需求分析

假设我们需要添加一个「竞争对手分析」功能：
- 分析历史中标单位
- 统计中标率
- 提供竞争策略建议

### 2.2 创建模块结构

```bash
src/
├── competitor/                    # 新模块目录
│   ├── __init__.py
│   ├── analyzer.py               # 分析器
│   ├── models.py                 # 数据模型
│   └── utils.py                  # 工具函数
```

### 2.4 添加 API 接口

编辑 `webapp/server_fastapi.py`：

```python
from src.competitor import CompetitorAnalyzer

# 初始化
competitor_analyzer = CompetitorAnalyzer(str(DB_PATH))

@app.get("/api/competitor/{buyer_name}")
async def get_competitor_analysis(buyer_name: str):
    """获取竞争对手分析"""
    analysis = competitor_analyzer.analyze_buyer(buyer_name)
    return {
        "buyer_name": analysis.buyer_name,
        "competitors": [
            {
                "name": c.name,
                "win_rate": c.win_rate,
                "avg_amount": c.avg_amount
            }
            for c in analysis.competitors
        ],
        "recommendations": analysis.recommendations
    }
```

## 四、添加单元测试

```python
# tests/test_competitor.py
import pytest
from src.competitor import CompetitorAnalyzer

def test_analyze_buyer():
    analyzer = CompetitorAnalyzer("data/ccgp_data.db")
    result = analyzer.analyze_buyer("某省政府办公厅")

    assert result.buyer_name == "某省政府办公厅"
    assert len(result.competitors) > 0
    assert all(c.win_rate >= 0 and c.win_rate <= 1 for c in result.competitors)
```
