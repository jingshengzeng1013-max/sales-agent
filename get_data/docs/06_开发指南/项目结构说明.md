## 二、核心模块详解

### 2.1 爬虫模块 (crawler)

**职责**：从中国政府采购网采集招标公告数据

**入口文件**：
- `ccgp_crawler.py` - 采集公告列表
- `crawl_detail.py` - 采集公告详情

**关键类/函数**：
```python
# ccgp_crawler.py
def crawl_list_pages(max_pages=100):
    """爬取列表页"""

# crawl_detail.py
def crawl_details(limit=None, test_first=False):
    """爬取详情页"""

# proxy_manager.py
class ProxyManager:
    def get_proxy(self) -> str:
        """获取代理"""
    def rotate_proxy(self):
        """轮换代理"""
```

**数据流**：
```
网站 → ccgp_crawler.py → tenders 表 → crawl_detail.py → tenders 表
                                              ↓
                                        attachments 表
```

### 2.2 ETL 模块 (etl)

**职责**：数据清洗、结构化抽取、项目聚合

**核心文件**：
- `core/extract_structured.py` - LLM 抽取
- `aggregate_projects.py` - 项目聚合

**关键类/函数**：
```python
# core/extract_structured.py
def extract_structured(limit=None, model="deepseek-chat"):
    """使用 LLM 提取结构化字段"""

# aggregate_projects.py
def aggregate_projects(structured_file, detail_file, output_file):
    """将同一项目的多条公告聚合"""
```

**数据流**：
```
tenders 表 → extract_structured.py → tenders_structured.jsonl
                                          ↓
                          aggregate_projects.py → projects.jsonl
```

### 2.3 检索模块 (retrieval)

**职责**：双路混合检索（向量 + 关键词）

**核心文件**：
- `retriever.py` - DualRetriever 类

**关键类**：
```python
class DualRetriever:
    def __init__(self, data_type="product"):
        """data_type: "product" 或 "tender" """

    def search(self, query, top_k=20, vector_weight=0.5, ...):
        """执行混合检索"""

    def _search_vector(self, query_vec, top_k):
        """向量检索"""

    def _search_bm25(self, query, top_k):
        """关键词检索"""

    def _rrf_fusion(self, vector_results, bm25_results):
        """RRF 融合"""
```

### 2.4 向量化模块 (vectorization)

**职责**：文本 Embedding、索引构建

**核心文件**：
- `vectorize_data.py` - 调用 Embedding API
- `build_index.py` - 构建 FAISS/BM25 索引

**关键函数**：
```python
def vectorize_tenders(input_file, output_file):
    """向量化招标数据"""

def build_faiss_index(embedded_file, index_file, id_map_file):
    """构建 FAISS 索引"""

def build_bm25_index(raw_data_file):
    """构建 BM25 索引"""
```

### 2.5 分析模块 (analysis)

**职责**：AI 销售建议、客户画像生成

**核心文件**：
- `sales_advisor.py` - SalesAdvisor 类
- `generate_customer_profiles.py` - 画像生成

**关键类**：

## 三、数据库表结构

### 3.1 tenders 表

原始招标公告表

```sql
CREATE TABLE tenders (
    id INTEGER PRIMARY KEY,
    project_name TEXT,
    publish_date TEXT,
    detail_url TEXT,
    content TEXT,
    attachment_urls TEXT,
    status TEXT DEFAULT 'new',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 3.2 tender_structured 表

结构化招标数据表

```sql
CREATE TABLE tender_structured (
    id INTEGER PRIMARY KEY,
    source_url TEXT,
    project_name TEXT,
    project_name_std TEXT,
    buyer_name TEXT,
    buyer_name_std TEXT,
    province TEXT,
    city TEXT,
    budget_amount REAL,
    announce_type TEXT,
    content_summary TEXT,
    technical_requirements_summary TEXT,
    product_keywords TEXT,
    contact_person TEXT,
    contact_phone TEXT,
    opportunity_score REAL,
    opportunity_reason TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 七、扩展开发指南

### 7.1 添加新的爬虫源

1. 在 `src/crawler/` 创建新文件
2. 参考现有爬虫实现
3. 添加配置项到 `config.py`

### 7.2 添加新的检索模型

1. 在 `src/retrieval/` 修改 `DualRetriever`
2. 或创建新的检索类
3. 更新 API 接口

### 7.3 添加新的分析功能

1. 在 `src/analysis/` 创建新模块
2. 参考 `sales_advisor.py` 实现
3. 添加 API 接口到 `webapp/server_fastapi.py`
