# 附件下载与解析流程

## 概述

本流程用于下载招标公告的附件文件（PDF/DOC/DOCX 等），解析文本内容后扩充到数据库，用于 RAG 检索。

## 流程步骤

```
1. 爬取附件链接 → 2. 下载附件 → 3. 解析附件 → 4. 导入数据库
```

## 第一步：爬取附件链接

```bash
# 爬取所有招标公告的附件链接
python -m src.crawler.batch_crawl_attachments

# 从 JSON 导入数据库
python -m src.crawler.batch_crawl_attachments --import
```

**输出：**
- `output/tenders_attachments.json` - 附件链接 JSON 文件
- 数据库表：`tender_attachments`

**说明：**
- 自动提取中国政府采购网的 bizDownload UUID 链接
- 生成实际下载 URL：`https://download.ccgp.gov.cn/oss/download?uuid=XXX`

## 第二步：下载附件

```bash
# 从数据库加载并下载所有附件
python -m src.crawler.download_attachments --db

# 或从 JSON 文件下载
python -m src.crawler.download_attachments --json output/tenders_attachments.json
```

**输出目录：**
```
attachments/
├── 54/
│   ├── 采购公告.pdf
│   └── 技术要求.docx
├── 55/
│   └── 招标文件.pdf
└── ...
```

**数据库更新：**
- `local_path` - 本地文件路径
- `file_size` - 文件大小
- `download_status` - 下载状态（downloaded）

## 第三步：解析附件文本

```bash
# 解析已下载的附件
python -m src.etl.parse_attachments
```

**依赖安装（可选）：**
```bash
# PDF 解析
pip install pypdf

# Word 文档解析
pip install python-docx

# Excel 解析
pip install openpyxl
```

**输出目录：**
```
output/parsed_attachments/
├── 54_采购公告.json
├── 54_技术要求.json
├── 55_招标文件.json
└── parsing_summary.json
```

**支持的格式：**
| 格式 | 解析库 | 说明 |
|------|--------|------|
| PDF | pypdf | 推荐 |
| DOCX | python-docx | Word 2007+ |
| DOC | - | 老版本需其他库 |
| XLSX | openpyxl | Excel 2007+ |
| CSV | 内置 | 内置支持 |
| TXT | 内置 | 内置支持 |

## 第四步：导入附件内容到数据库

```bash
# 导入解析后的附件内容
python -m src.etl.import_attachment_chunks --import

# 验证导入结果
python -m src.etl.import_attachment_chunks --verify
```

**数据库表：tender_chunks**
- `chunk_type` = "attachment_content"
- `chunk_text` - 附件文本分块
- `metadata_json` - 源文件信息

## 完整流程示例

```bash
# 1. 爬取附件链接
python -m src.crawler.batch_crawl_attachments
python -m src.crawler.batch_crawl_attachments --import

# 2. 下载附件
python -m src.crawler.download_attachments --db

# 3. 解析附件
python -m src.etl.parse_attachments

# 4. 导入数据库
python -m src.etl.import_attachment_chunks --import

# 5. 验证
python -m src.etl.import_attachment_chunks --verify
python tests/utils/verify_data.py
```

## 数据流向

```
中国政府采购网
    ↓
HTML 详情页（bizDownload 区域）
    ↓
UUID 提取 → 下载链接生成
    ↓
tender_attachments 表
    ↓
本地下载（attachments/）
    ↓
文本解析（parsed_attachments/）
    ↓
tender_chunks 表（chunk_type=attachment_content）
    ↓
RAG 向量检索
```

## 常见问题

### 1. 附件下载失败
- 检查网络连接
- 部分链接可能有过期时间
- 重新爬取最新链接

### 2. PDF 解析乱码
- 部分 PDF 为扫描版，需要 OCR
- 可后续集成 pdfplumber 或 pytesseract

### 3. 分块大小调整
在 `import_attachment_chunks.py` 中修改：
```python
chunks = chunk_text(text, chunk_size=500, overlap=50)
```

### 4. 大量附件处理
建议分批处理：

```bash
# 可按 tender_id 范围分批
```
