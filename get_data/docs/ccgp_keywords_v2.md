## 关键发现

1. **卫星移动通信关键词在CCGP数据偏少**：仅"应急救援设备"(58条)、"应急通信"(17条)、"卫星通信"(11条)、"卫星电话"(16条)有有效数据。"天通"相关词在CCGP几乎无收录，建议用"应急通信"、"卫星电话"、"应急救援设备"等替代。

2. **工业5G/AI关键词效果最佳**：机器人(215条)、人工智能(181条)、AI(285条)、专网(76条)、智能制造(77条)均有大量数据，是政采重点方向。

3. **星闪短距通信在CCGP暂无数据**：政采中对"星闪"技术的采购尚未形成规模，建议持续监控，并通过"无线投屏"(10条)等替代词寻找相关采购。

4. **应急通信场景关键词量最大**：应急(1630条)、消防(1734条)、救援(957条)等宽泛词覆盖量极大，需精匹配过滤；"应急救援"(159条)、"森林防火"(80条)等精确词更有针对性。

5. **卫星通信芯片/模组/终端等精确产品词无数据**：CCGP政采中通常不直接采购芯片，而是采购终端整机。建议采集时以"卫星通信"、"应急通信设备"、"卫星电话"等产品形态词为主。
